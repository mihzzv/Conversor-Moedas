# Conversor-Moedas
Aplicação web desenvolvida para conversão de moedas em tempo real, permitindo converter valores em Real (BRL) para diferentes moedas internacionais, como Dólar, Euro, Libra Esterlina, Iene Japonês, Peso Argentino e Dólar Canadense. O projeto consome dados de câmbio por meio de API, oferecendo uma interface moderna e responsiva desenvolvida com HTML, CSS e JavaScript.
